Concepto General

CSS son las siglas de "Cascade StyleSheet". Se trata de
una especificación sobre los estilos físicos aplicables a un documento HTML, y trata de dar la separación
definitiva de la lógica (estructura) y el físico (presentación) del documento.

Objetivos

  • Dar formato a páginas web
  • Separar diseño de contenido
  • Centralizar propiedades de aspecto
  • Facilitar mantenimiento de diseño

Características

  • Fichero de texto
  • Formato de páginas HTML, PHP...
  • Herencia de propiedades entre elementos
  • Clases definibles por el usuario
